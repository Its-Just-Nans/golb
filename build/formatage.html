Partition and format on linux

Know type of partition

df -Th

Legend :

  • -T : --print-type
  • -h : --human-readable

Format partition

fdisk /dev/sda

# touch d for delete partition

# touch n for new partition

Format partition

mkfs.vfat /dev/sda1

Use dd to copy an ISO

dd if=file.iso of=/dev/sda bs=1M status=progress

Legend :

  • if : input file
  • of : output file
  • bs : block size
